# Hand-drawn slots

Every decorative element on the site lives here as a named file. To make the
site more *hers*, Louella draws a replacement on paper, you scan it, clean it
up (transparent background, dark ink), and save it over the placeholder with
the same filename. The site picks it up on the next build — no code changes.

Current slots:

| File          | Where it appears              | Suggested size      |
| ------------- | ----------------------------- | ------------------- |
| `divider.svg` | Footer + section breaks       | wide & short (~220×12) |
| `splat.svg`   | Home page accent doodle       | roughly square      |

Ideas for future slots (add the file, then wire it in): logo wordmark, nav
icons, 404 illustration, "the end" stamp for novel pages, corner doodles.

Tips for scans → SVG/PNG: photograph in bright light, increase contrast, and
use a tool like Vectorizer or Inkscape's Trace Bitmap for crisp SVG lines.
PNG with transparency works fine too — just keep the same filename stem and
update the import extension.
